# 🔌 Z-ARTIST VAULT — Plugin Guide
> Z-Artist writes in images first, words second. These plugins treat the vault like a studio wall.

---

## ESSENTIAL INSTALLS

### 1. Obsidian Canvas (Built-in)
**What it does:** Infinite 2D whiteboard. Place notes, images, cards, and arrows anywhere. Native to Obsidian 1.1+.
**Enable:** Settings → Core Plugins → Canvas ✓

**Shortcut:** `Cmd+Shift+C`

**How Z-Artist uses it:**
- Before writing a Mural Bar sequence, sketch the 8-panel visual on Canvas
- Each canvas card = one bar's image. Arrange spatially → the spatial arrangement IS the bar order
- Import photos of chalk murals (or reference imagery) → annotate with lyric ideas
- The "Wounded Rose" Canvas: a permanent reference board for Zed's signature motif across tracks

**Workflow:**
1. `Cmd+Shift+C` → new canvas → "Track Name — Visual"
2. Place 8 blank cards in sequence
3. Write a single image word in each (CROWD / GREY / FOUNTAIN / ZED / CHALK / MURAL / DUST / SILENCE)
4. Now write the bars. The images are already in order.

> **This is the Z-Artist difference.** K-9 and Dog-Prophet write *at* the page. Zed builds it visually first.

---

### 2. Excalidraw
**What it does:** Freehand drawing, diagrams, and visual thinking inside Obsidian notes.
**Install:** Search "Excalidraw"

**Shortcut:** `Cmd+Shift+D`

**How Z-Artist uses it:**
- Sketch the mural layout *before* writing the lyric
- The mural IS the lyric. Draw it. Then translate it.
- Dual-panel: Excalidraw sketch on left, Longform lyrics on right
- Twin study diagrams: draw the Zed/Rax duality visually — where they overlap, where they split

---

### 3. Annotator
**What it does:** Annotate PDFs and images directly inside Obsidian. Highlight, comment, tag sections.
**Install:** Search "Annotator"

**How Z-Artist uses it:**
- Screenshot a completed chalk mural → import → annotate each section with the bar it inspired
- Annotate reference grime lyrics (Skepta, Stormzy, JME) directly — tag what Z-Artist's mural bars owe and what they break from
- Build a visual annotation layer over the SD Media discography PDF

---

### 4. Gallery
**What it does:** Turns a folder of images into a visual gallery inside a note.
**Install:** Search "Obsidian Gallery"

**How Z-Artist uses it:**
- `Canvas/Mural-Archive/` folder → gallery view → see all chalked murals as thumbnails
- Browse visually, not by filename
- Drag from gallery into Canvas for reference boards

---

### 5. Image Toolkit
**What it does:** Click any image in a note → zoom, rotate, drag, resize. Full image interaction.
**Install:** Search "Image Toolkit"

**How Z-Artist uses it:**
- Working from a mural reference image → zoom into a specific section → write the corresponding bar
- No need to leave the note

---

### 6. Dynamic Highlights
**What it does:** Highlights custom word patterns across the vault.
**Install:** Search "Dynamic Highlights"

**Z-Artist highlight patterns:**

| Pattern | Color | Why |
|---------|-------|-----|
| `Rax` | Cyan `#44ffee` | Every Rax mention is a Zed ghost |
| `mask\|masked` | Grey | Track the mask motif |
| `chalk\|mural\|Wounded Rose` | Blue | Visual art vocabulary |
| `scar\|scarred\|stabbed\|truck` | White | Body-as-text references |
| `Russian\|accent` | Red | The cover. Whenever it appears. |
| `\[MURAL BAR\]` | Blue | Flag Mural Bar sequences |

---

### 7. Longform
**What it does:** Verse-by-section writing environment with compile.
**Install:** Search "Longform"

**Z-Artist config:**
- Scenes: `ESTABLISH / DETAIL-1 / DETAIL-2 / ZOOM-1 / ZOOM-2 / ZOOM-3 / REFRAME / LANDS`
- This maps directly to the Mural Bar 8-panel structure
- Every Longform scene = one brushstroke

---

### 8. Footnotes
**What it does:** Inline footnotes linked to bottom of note.
**Install:** Search "Obsidian Footnotes"

**Z-Artist use:**
```
I drag my chalk across the face of things[^1]

[^1]: Literal: mural on concrete. Layer 2: Zed marking his presence where he's not supposed to be. Layer 3: "face of things" = Rax's unscarred face. The twin parallel.
```

---

### 9. Cycle Through Panes
**What it does:** `Alt+Tab` through open panes without using mouse.
**Install:** Search "Cycle Through Panes"

**Z-Artist use:**
- Canvas on left, Excalidraw sketch in middle, Longform lyrics on right
- Cycle between them with keyboard while keeping hands on the draft
