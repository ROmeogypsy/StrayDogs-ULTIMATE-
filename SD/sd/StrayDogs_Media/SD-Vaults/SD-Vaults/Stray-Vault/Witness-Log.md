# 👁️ WITNESS LOG
> *Stray sees everything. This is where it lives before it becomes a bar.*

## HOW TO USE
- Write in present tense. Immediate. Sensory.
- No editorializing. Just what's seen.
- Every entry is dated. Every entry is potential source material.
- When an entry becomes a bar: tag it `#used` and link to the track.
