# 🔌 STRAY VAULT — Plugin Guide
> Stray doesn't write hot. These plugins are built for observation, negative space, and the slow pivot.

---

## ESSENTIAL INSTALLS

### 1. Excalidraw
**What it does:** Full whiteboard/drawing canvas inside Obsidian. Freehand. Diagrams. Visual brainstorming.
**Install:** Search "Excalidraw"

**Shortcut:** `Cmd+Shift+D`

**How Stray uses it:**
- Map the emotional geography of a track before writing a word
- Draw the *scene* the bar takes place in — where is Stray standing? What can he see?
- Use it to sketch the Pivot: draw the "seems to be about" as one shape, the "actually lands on" as another — find the bridge between them
- Mural-reference boards for Z-Artist collabs (screenshot Zed's chalk work → paste into canvas → annotate with lyric ideas)

> **Why Stray needs this over K-9:** K-9 fires. Stray *watches*. Before a single word, Stray needs to see the scene. Excalidraw is the scene.

---

### 2. Journey
**What it does:** Finds the connection path between any two notes in your vault. Like "Six Degrees of Separation" but for ideas.
**Install:** Search "Journey"

**Shortcut:** `Cmd+Shift+J`

**How Stray uses it:**
- "Find path from [[Big 9]] to [[The Fountain]]" → it shows you every linked note in between
- This *is* the Stray method: the long route between two things that shouldn't connect — that's where the pivot lives
- Use it to find a lyrical bridge when a verse feels stuck

> **Why Stray needs this:** The Pivot isn't a trick — it's a discovered relationship. Journey makes those relationships visible.

---

### 3. Sliding Panes (Andy's Mode)
**What it does:** Notes stack horizontally like cards. Click a link → it opens beside the current note instead of replacing it. Build a reading trail.
**Install:** Search "Sliding Panes" or "Andy's Mode"

**Shortcut:** `Cmd+Shift+P` to toggle

**How Stray uses it:**
- Open the Witness Log → link to a track → link to a bar analysis → see all three side by side
- The visual trail of connected notes mirrors Stray's writing process: one thing leads to another leads to the pivot

---

### 4. Enhancing Mindmap
**What it does:** Turns any bullet-point note into a live interactive mindmap. Click a node, edit the note, map updates.
**Install:** Search "Enhancing Mindmap"

**How Stray uses it:**
- Outline a verse as bullets → toggle mindmap view → see the structure visually
- Great for the Silence Study — map what's *not* being said as nodes

---

### 5. Footnotes
**What it does:** `Cmd+Shift+6` inserts a footnote anywhere. Numbered, linked, shown at bottom.
**Install:** Search "Obsidian Footnotes"

**How Stray uses it:**
```
I move between postcodes like water[^1]

[^1]: Layer 2: rootlessness. No fixed identity. Layer 3: the literal route between the Doghouse and the Yard that Stray walks when he's observing.
```

> **Why Stray specifically:** Every Stray bar has sub-surface meaning. Footnotes let you annotate the layers directly in the lyric without cluttering it.

---

### 6. Reading Time
**What it does:** Estimates reading/speaking time for any note. Shows in status bar.
**Install:** Search "Reading Time"

**How Stray uses it:**
- A verse that reads in 0:18 when spoken at natural cadence is the right length
- Stray's drift cadence runs slower than K-9's blast — reading time helps calibrate

---

### 7. Random Note
**What it does:** Opens a random note from the vault.
**Install:** Built into core plugins — enable in Settings → Core Plugins

**Shortcut:** `Cmd+Shift+R`

**How Stray uses it:**
- Stuck on a pivot? Random note. Read whatever comes up. Find the unexpected connection.
- This is a Stray-specific technique: observation without intention. See what appears.

---

### 8. Longform
**What it does:** Verse-by-section writing with drag-to-reorder and compile.
**Install:** Search "Longform"

**Stray-specific use:**
- Use "Scenes" but name them differently: `Before / The Turn / After`
- The Pivot is always Scene 2. Everything else is setup or landing.
