## How to Use This Page

1. Drop a bar in the **Input** section
2. Work through all three layers
3. If it has three solid layers, it passes. Move it to **Cleared Bars**.
4. If it only has two, note what's missing and keep working.
5. See [[Techniques/Syllabic-Embedding]] for method reference.
