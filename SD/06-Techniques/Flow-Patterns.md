# Flow Patterns & Cadence Schemes
**Tags:** #technique #flow #craft

---

## The Four SD Media Flow Archetypes

### 🔥 The K-9 Blast
- **Pattern:** Short, punchy bars. End-rhyme heavy. Builds in intensity across the verse.
- **Rhythm:** Fills all the space. No breath until the verse drops.
- **Best for:** Hooks, openers, confrontational verses, hype.
- **Pitfall:** Can run out of steam if every bar hits at the same level. Need a gear shift.
- **Example structure:** `A A B B / C C D D / [ESCALATE] / A' A' B' B'`

---

### 🌫️ The Stray Drift
- **Pattern:** Meandering lines that seem to wander — then land. Pauses are intentional.
- **Rhythm:** Sits behind the beat. Off-the-grid cadence.
- **Best for:** Storytelling, reflection, the verse that recontextualizes the hook.
- **Pitfall:** Can feel like it's going nowhere. Needs a landing bar that earns the drift.
- **Example structure:** `Long A... B... Long A... [PIVOT] C / C / C HARD STOP`

---

### 🖼️ The Z-Artist Mural
- **Pattern:** 8-bar unit that functions as a single image. Internal rhyme scheme, not end-rhyme dominant.
- **Rhythm:** Deliberate. Measured. Each bar is a brushstroke.
- **Best for:** Album tracks, deep cuts, the verse everyone cites.
- **Pitfall:** Can feel academic without a visceral image at the center.
- **Example structure:** `[IMAGE ESTABLISH] / [DETAIL] / [DETAIL] / [ZOOM] / [ZOOM] / [ZOOM] / [REFRAME] / [FULL IMAGE LANDS]`

---

### 📜 The Dog-Prophet Commandment
- **Pattern:** Statement. Statement. VERDICT. Repeat.
- **Rhythm:** Preacher cadence. Pauses where the crowd would respond.
- **Best for:** Closers, bridges, the moment a track gets serious.
- **Pitfall:** Can become monotonous. Needs variation in the verdict delivery.
- **Example structure:** `[OBSERVATION] / [OBSERVATION] / [COMMANDMENT] / silence / [OBSERVATION] / [ESCALATION] / [FINAL COMMANDMENT]`

---

## Hybrid Flows (Collab Patterns)

### The Transfer
One writer's cadence seamlessly transitions to another mid-verse. The seam should be invisible.

### The Response
Writer B directly answers Writer A's last bar. Best in back-and-forth structures.

### The Contrast Cut
Writers with opposing energy — K-9 blast into Stray drift, for example. The contrast IS the technique.

## Cadence Reference Chart

| Term | Description |
|------|-------------|
| On the beat | Stresses align with kick/snare |
| Ahead of the beat | Rushing — creates urgency |
| Behind the beat | Laid back — creates atmosphere |
| Triplet flow | Three syllables per beat subdivision |
| Polyrhythmic | Two rhythmic patterns simultaneously |
| Stop-start | Deliberate gaps for effect |
