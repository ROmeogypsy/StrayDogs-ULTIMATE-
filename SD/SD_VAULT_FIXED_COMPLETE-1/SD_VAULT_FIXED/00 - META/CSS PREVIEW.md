## 🎯 SPECIAL CLASSES

You can use custom classes for extra styling:

**Track lyrics** — Add this to any paragraph:
```html
<div class="track-lyrics">
Your lyrics here
Gold left border, graffiti font, spaced out
</div>
```

**Character dialogue colors:**
- `.dialogue-angel` → Neon red, bold
- `.dialogue-stray` → Neon green, italic
- `.dialogue-k9` → Royal blue, bold
- `.dialogue-dame` → Gold

**Tension markers:**
- `.tension-critical` → Red, pulsing, uppercase
- `.tension-high` → Orange, bold

**Dog number badges:**
- `.dog-number` → Gold rounded badge with shadow
